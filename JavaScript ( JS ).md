# JavaScript ( JS )

#### JS 란 ? HTML 문서를 동적으로 변경할 수 있는 언어

- 특정 상황에서 html 문서를 다르게 (동적) 보여주기 위한 언어
- HTML 이 없다면 JS는 무용지물



- #### var

  - 생략 가능
  - 변수 호이스팅
  - 암묵적 결합(중복 선언 가능)
  - 함수 레벨 스코프

- #### let

  - 생략 불가능
  - 변수 호이스팅이 동작하지 않는 것 처럼 동작
  - 재선언 불가 ( 재할당은 가능 )
  - 블록 레벨 스코프

- #### const 

  - 생략 불가능
  - 변수 호이스팅이 동작하지 않는 것 처럼 동작
  - 재선언 불가 (재할당 불가)
  - 블록 레벨 스코프
